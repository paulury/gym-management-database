# gym-management-database
# Base de données – Gestion d'une salle de sport

Projet universitaire visant à concevoir une base de données complète pour gérer les opérations d'une salle de sport.

## Objectif
Créer une base de données relationnelle permettant la gestion des membres, coachs, cours et réservations.

## Technologies utilisées
- SQL
- MySQL Workbench

## Fonctionnalités

### Interface gestionnaire
- Gestion des membres et adhésions
- Gestion des coachs
- Gestion des cours et capacités
- Gestion des réservations

### Interface membre
- Inscription
- Réservation de cours
- Consultation de l’historique

### Analyse statistique
- Participation aux cours
- Analyse popularité des coachs
- Statistiques d’inscriptions

## Conception technique
- Diagramme Entité/Association
- Création complète des tables
- Peuplement de la base
- Requêtes avancées :
  - Sous-requêtes
  - Jointures SQL
  - Fonctions d’agrégation
  - Requêtes ensemblistes

## Contraintes
Gestion des capacités de cours et gestion des droits administrateurs.
